# README: PikaMovie App

## Setting up
To create this project, we used:
* Ruby 2.6.3p2
* Rails 6.0.2.2
* React
* Devise
* Jest/Enzyme
* Rspec (rspec-rails 4.0.0)

For you to use this project, you should run:
* rails db:create
* rails db:migrate
* yarn
* bundle

To implement the tests, you should run:
* yarn test (runs Jest)
* rsepc spec (runs Rspec)
